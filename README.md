# SaaS Landing Page with animations

Design SaaS Landing Page with animations.

## Tech Stack

- **Next JS**
- **Tailwind**
- **Typescript**
- **Framer Motion**

First, run the development server:

```bash

yarn dev
# or
pnpm dev
# or
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
